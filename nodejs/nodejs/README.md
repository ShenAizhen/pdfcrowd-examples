# Pdfcrowd & Node.js Web Server Demo

Sample for dynamic and static HTML to PDF conversions.

### Prerequisites

```
nodejs
npm
```

Replace **'your_username'** with your Pdfcrowd username and **'your_apikey'** with your Pdfcrowd API key in **[server.js](server.js#L49)** file. Sign up for a [free trial](https://pdfcrowd.com/user/sign_up/?pid=api-trial2) if you don't have the credentials.

## Installing

```
npm install
```

## Web server start

```
nodejs server.js
```

## Test

   Open a sample web page <http://localhost:8080/>

   Invoke a conversion to PDF by pressing buttons at the bottom of the page.
